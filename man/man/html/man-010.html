レイヤーウィンドウ

このウィンドウは本ツールで前面に来ている主ウィンドウで編集している画像の「レイヤーの管理」を行います。


構造

レイヤー構造はリストで表示します。リストの最後が最下層のレイヤーとなるように表示します。

このリストでは次の操作が行えます（編集機能の詳細はそれぞれの説明文にあるリンク先を参照してください）。


順序移動 リストでドラッグして順序を入れ替えます。 説明
複製 コントロールキーあるいは副ボタン（右ボタン）を押しながら順序移動の操作を行うとレイヤーを複製します。 説明
表示 レイヤーの表示を切り替えます。 説明
保護 レイヤーの編集の可否を切り替えます。 説明
名称 レイヤーの名称を編集します。 説明


生成・複製・削除・統合

レイヤーウィンドウに配置するボタンは、次の機能を実行するためのものです（編集機能の詳細はそれぞれの説明文にあるリンク先を参照してください）。


生成 新規レイヤーを最前面に生成（追加）します。 説明
複製 選択しているレイヤーの真上にレイヤーを複製します。 説明
削除 選択しているレイヤーを削除します。 説明
統合 表示しているレイヤーを統合します。 説明
