パレット

パレットに関する操作をまとめたメニューで、個々のメニューはパレットウィンドウに配置する各ボタンや編集操作とほぼ等価な機能です。


補助操作
固定 ボタンと等価です。 説明
ウィンドウ拡縮 と等価です。 説明
色選択
次の色へ パレットウィンドウで選択している色選択番号を一つ次のものに切り替えます。
前の色へ パレットウィンドウで選択している色選択番号を一つ前のものに切り替えます。
下のレイヤーを使用 描画時の色（カラーパターン）として背面レイヤーを使用します。パレットウィンドウの左上隅のクリックと等価です。
色混合
色混合は RGB で演算 色混合を RGB にします。パレットウィンドウでRGBを選択することと等価です。
色混合は HLS で演算 色混合を HLS にします。パレットウィンドウのHLSを選択することと等価です。
編集
色補助属性を設定 ボタンと等価です。 説明
パレットグラデーション ボタンと等価です。 説明
色を交換 ボタンと等価です。 説明
色を複写 ボタンと等価です。 説明
色要素操作
すべてを +1 選択している色の赤成分（R）、緑成分（G）、青成分（B）の値をすべて +1 します。
赤を +1 選択している色の赤成分（R）の値を +1 します。
色要素で１の位を主ボタンクリックした場合と等価です。

色要素で１の位を主ボタンクリックした場合と等価です。

色要素で１の位を主ボタンクリックした場合と等価です。
