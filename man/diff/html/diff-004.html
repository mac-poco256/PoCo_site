Version 1.300(1.3.0) から Version 1.310(1.3.1) の変更点

Version 1.300(1.3.0) から Version 1.310(1.3.1) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • テクスチャを追加しました。
  • 塗り選択で、色の範囲（連続数）を指定する機能を追加しました。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • レイヤー名称を変更しようとすると、Out of Range 例外が発生しアプリケーションが強制終了する不具合を修正しました。
  • 境界色の指定を伴った塗り選択の処理速度を改善しました。
  • レイヤーウィンドウが前面にある場合、すべてを選択メニューが作動しない不具合を修正しました。


削除（廃止）

削除、ないし廃止の機能はありません。
