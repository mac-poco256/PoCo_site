Version 1.330(1.3.3) から Version 1.340(1.3.4) の変更点

Version 1.330(1.3.3) から Version 1.340(1.3.4) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • パレットウィンドウの表示位置と選択色を同期する機能を追加しました。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • 色要素を変更した場合、表示ウィンドウの表示位置が中心に移動する不具合を修正しました（Version 1.330(1.3.3) で混入した不具合です）。


削除（廃止）

削除、ないし廃止の機能はありません。
