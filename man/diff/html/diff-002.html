Version 1.100(1.1.0) から Version 1.200(1.2.0) の変更点

Version 1.100(1.1.0) から Version 1.200(1.2.0) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • レイヤープレビューの大きさを設定する機能を追加しました。
    この機能追加に伴い、レイヤーウィンドウの高さを変更する際に課していた制限を撤廃しました（一定の刻みでしか変形できないようにしていましたが、任意の大きさを可能としました）。
  • 色置換で色の選択範囲（選択数）を指定する機能を追加しました。
  • 彩度無しを透明度に読み替えて保存する機能を追加しました。
  • 自動グラデーションで隣接を指定している場合には、個別の色の組み合わせごとにサイズを指定する機能を追加しました。
    また、このサイズはその色の画素数の総和に応じた値を初期値として割り振ります。
  • 塗り選択の機能に境界色を指定した選択をする機能を追加しました。
  • 色設定シートに CMYK モードを追加しました。
    ただし、これは色の選択方法に CMYK を追加したのみであり、色空間は従来どおりに RGB です。
  • 新しいタブを追加する際の動作を追加しました。編集中の画像と同じサイズの新規ドキュメント（新規画像）を作成します（この動作は macOS 10.12 Sierra 以降でのみ作用します）。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • 色混合を用いた塗りつぶしでグラデーションをかける場合、パターンの反転の指定に合わせて濃度の始点を入れ替えるように変更しました。
  • 自動グラデーションの色選択の初期値は、選択範囲内にある色に基づくように変更しました（従来は選択範囲内の色には依らずに、パレットの設定にのみ依存した初期値としていました）。
  • 消去を行う際、パレット先頭の色がマスク（上塗り禁止）だと消去が実行できない不具合を修正しました。


削除（廃止）

削除、ないし廃止の機能はありません。
