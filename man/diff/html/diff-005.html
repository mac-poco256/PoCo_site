Version 1.310(1.3.1) から Version 1.320(1.3.2) の変更点

Version 1.310(1.3.1) から Version 1.320(1.3.2) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • 表示倍率のスライダーのステップ数を追加しました。
  • パレットウィンドウに上下の矢印ボタンを（渋々ながらに）追加しました。
  • 主ウィンドウ内の四方アキは、画面サイズの１０％に変更しました。また、表示倍率を変更した場合の表示位置の再計算の方法を改良しました。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • スペースキーを押すと Undo/Redo が動作しなくなることがある不具合を修正しました。
  • レイヤー補助属性を編集すると、選択レイヤーが不意に切り替わることがある不具合を修正しました。


削除（廃止）

削除、ないし廃止の機能はありません。
