Version 1.000(1.0.0) から Version 1.100(1.1.0) の変更点

Version 1.000(1.0.0) から Version 1.100(1.1.0) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • 自動グラデーションの機能を追加し、メニューに同機能の呼び出し項目を追加しました。
  • 色置換の機能を追加し、メニューに同機能の呼び出し項目を追加しました。
  • メニューにペン太さを太くする、ペン太さを細くするを追加しました。
  • メニューに濃度を濃くする、濃度を薄くするを追加しました。
  • 色複写の機能を追加し、パレットウィンドウに同機能の呼び出しボタンを、メニューに同機能の呼び出し項目を追加しました。
  • レイヤープレビューの表示品質を設定する機能を追加しました。
  • 自由曲線および筆圧比例自由曲線で、ペン先を消しゴムに切り替えなくても、消しゴム相当の機能で描画する機能を追加しました。
  • 塗り選択の機能を追加しました。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • retina 解像度への対応を行いました。
  • Bundle Identifier の名称を変更しました（必要であればこちらを参考に UserDefaults の引き継ぎを行ってください）。
  • パレット固定化のアイコンを変更しました。
    色要素の編集可否を表すアイコンになります。
  • 連続／不連続線のアイコンを変更しました。
    支点間を結ぶか否かを表すアイコンになります。
  • ハンドル使用のアイコンを変更しました。
    ハンドルの使用不使用を表すアイコンになります。
  • 範囲選択のアイコンを変更しました。
    矩形形状か任意形状かを表すアイコンになります。
  • レイヤー編集可否の項目名（テーブルヘッダの表示文字列）を編集から保護へ変更しました。
  • 半値の霧吹きサイズ４のパターンがサイズ３と同じになっていた不具合を修正しました。
    また、この修正に伴わせてサイズ４のパターンの見直しを行いました。


削除（廃止）

削除、ないし廃止の機能はありません。
