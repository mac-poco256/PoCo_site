Version 1.320(1.3.2) から Version 1.330(1.3.3) の変更点

Version 1.320(1.3.2) から Version 1.330(1.3.3) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • 主ウィンドウの表示位置と表示ウィンドウの表示位置を同期する機能を追加しました。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • レイヤーの表示・非表示を切り替えた後に表示をスクロールすると、正しくない表示を行っていた不具合を修正しました。
  • 背景が透明の場合に縮小表示を行うと、正しくない表示を行っていた不具合を修正しました（Version 1.320(1.3.2) で混入した不具合です）。


削除（廃止）

削除、ないし廃止の機能はありません。
