Version 1.200(1.2.0) から Version 1.300(1.3.0) の変更点

Version 1.200(1.2.0) から Version 1.300(1.3.0) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能は次のとおりです。

  • 補間曲線を描画する機能を追加しました。
    また、補間方法を選択する機能を追加しました。
  • 色補助属性の一括指定で色の選択範囲（選択数）を指定する機能を追加しました。
  • 任意角回転で、矩形および楕円を描画する機能を追加しました。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • 色補助属性で非使用としている色の場合、ぼかしの演算対象から除外するように変更しました。
  • 画像の選択範囲を削除する場合は、選択している描画系の機能を反映するように機能を変更しました。ただし、濃度と均一濃度は同じ結果となります。
  • メニューの項目名を消去から削除へ変更しました。
  • 形状を使用した、ぼかしの描画範囲が１ドット大きくなっていた不具合を修正しました。


削除（廃止）

削除、ないし廃止の機能はありません。
