Version 1.350(1.3.5) から Version 1.360(1.3.6) の変更点

Version 1.350(1.3.5) から Version 1.360(1.3.6) へ更新するにあたり、変更となった内容についてまとめています。


追加

新規に追加した機能はありません。


変更（訂正）

変更、ないし訂正した機能は次のとおりです。

  • macOS 10.14.x に於いて、パレット詳細リストが正常にスクロールしなくなっていた不具合を修正しました。
  • macOS 10.14.x に於いて、About panel が開かない不具合を修正しました。
  • 特定の条件の下では自動グラデーションが正常に動作しないことがある不具合を修正しました。


削除（廃止）

削除、ないし廃止の機能はありません。
