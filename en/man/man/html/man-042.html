The Continuous Line

The button indicate that discontinuous line, which modify drawing function as following:

  • In the free line and the free line with the pen pressure, between each point does not draw(no-continuous line) when the main button drag.
  • In shapes(including rectangle, ellipse, and parallelogram), between each control point does not draw(no-continuous line)
  • In the selecion, the shape of selection is always rectangle.(the button is indicated.)


The button indicate that continuous line, which modify drawing function as following:

  • In the free line and the free line with the pen pressure, between each point draw(continuous line) when the main button drag.
  • In shapes(including rectangle, ellipse, and parallelogram), between each control point draw(continuous line)
  • In the selecion, the shape of selection is the free style.(the button is indicated.)
