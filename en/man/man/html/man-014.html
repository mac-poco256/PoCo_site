The Edit Menu

The edit menu relative to undo and select.


Relative to Undo
Undo Undo.
Redo Redo.
Relative to Selection
Copy Selected image in current layer copy to the Paste Board.
Paste Copy from the Paste Board to current layer(that's, Paste).
Delete Fill setected image in current layer
Select All Select all on current layer.
Select Clear Cancel select, so that there is not selected area.
Clip Image Clip image with rectangle selection area.
Flip Horizontal Selected image in current layer is flipped by horizontal.
Flip Vertical Selected image in current layer is flipped by vertical.
Register a Color Pattern Register selected image in current layer to the Color Pattern.
Auto Gradient The auto gradient process apply to selected image in current layer.
Replace Color The color replace process apply to selected image in current layer.
