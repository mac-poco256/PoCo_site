The Layer Window

This window works that manages the layer structure of current document.


Layer Structure, Layer List

The layer structure is represented by list. An entry at most bottom within list is most bottom layer.

The following functions are able to do on this list: (for details of each function, see to their description (access to description via link).)


Reordering An entry of list allow to drag to be reordered.

desc.
Duplicate (Copy) This operation is similar describe below 'Reordering'. If sub button (right button) on mouse is pressed when an entry of list drop to target layer, the dragged layer is duplicated (copied). desc.
Visible Switch either visible or invisible. desc.
Protection Switch either allow or disallow to be drawn. desc.
Name Edit layer name. desc.


Create, Delete and Unify

Some buttons are placed for the following functions (for details of each function, see to their description (access to description via link).)


Create Create layer with current colour at most top of list. desc.
Copy Copy the selected layer and place it right above. desc.
Delete Delete the setected layer. desc.
Unify Unify with all the visible layer into a single layer. desc.
