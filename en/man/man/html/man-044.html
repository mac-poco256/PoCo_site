The Control Point and Identical Shape

This function specify a type of moving of control point when drawing frame or filling


 Fixed Original Point

This button indicates that fix the start point.(i.e. any point share the start point.)


 Follow Last Point

This button indicates that move the start point to the last point.


 Identical Shape

This button indicates that keep shape, which move all control point(include relay point). However, this function does not work in interpoloation.
