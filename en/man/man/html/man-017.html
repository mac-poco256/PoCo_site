The Palette Menu

The palette menu is the gruop of operate of palette manage, which behavior of each menu is similar to each button on the palette window and operate.


Auxiliaries
Lock Same as button. desc.
Collapse Same as button. desc.
Select of Color
Next Color The index of current selected color on the palette window change to next index.
Previous Color The index of current selected color on the palette window change to previous index.
Below Layer Indicate drawing with a below layer instead of a color pattern. This menu behavior is same as click left top corner on the palette window.
Color Mixing Mode
RGB Mixing The mode of color mixing indicate to RGB. This menu behavior is same as select of RGB on the paltte window.
HLS Mixing The mode of color mixing indicate to HLS. This menu behavior is same as select of HLS on the paltte window.
Edit
Auxiliary Setting Same as button. desc.
Create Palette Gradient Same as button. desc.
Exchange Color Same as button. desc.
Copy Color Same as button. desc.
Color Elements and Auxiliary Attributes
Increase All By One All element of current color(R(red), G(green) and B(blue)) are increased by one.
Increase Red By One The R(red) element of current color is increased by one.
This menu behavior is same as click the ones digit on the detail of color element.
Increase Green By One The G(green) element of current color is increased by one.
This menu behavior is same as click the ones digit on the detail of color element.
Increase Blue By One The B(blue) element of current color is increased by one.
This menu behavior is same as click the ones digit on the detail of color element.
Decrease All By One All element of current color(R(red), G(green) and B(blue)) are decreased by one.
Decrease Red By One The R(red) element of current color is decreased by one.
This menu behavior is same as secondary click the ones digit on the detail of color element.
Decrease Green By One The G(green) element of current color is decreased by one.
This menu behavior is same as secondary click the ones digit on the detail of color element.
Decrease Blue By One The B(blue) element of current color is decreased by one.
This menu behavior is same as secondary click the ones digit on detail of color element.
Increase All By 16 All element of current color(R(red), G(green) and B(blue)) are increased by sixteen.
This menu behavior is same as click the color sample.
Increase Red By 16 The R(red) element of current color is increased by sixteen.
This menu behavior is same as click tens digit on the detail of color element.
Increase Green By 16 The G(green) element of current color is increased by sixteen.
This menu behavior is same as click tens digit on the detail of color element.
Increase Blue By 16 The B(blue) element of current color is increased by sixteen.
This menu behavior is same as click tens digit on the detail of color element.
Decrease All By 16 All element of current color(R(red), G(green) and B(blue)) are decreased by sixteen.
This menu behavior is same as click the color sample.
Decrease Red By 16 The R(red) element of current color is decreased by sixteen.
This menu behavior is same as secondary click tens digit on the detail of color element.
Decrease Green By 16 The G(green) element of current color is decreased by sixteen.
This menu behavior is same as secondary click tens digit on the detail of color element.
Decrease Blue By 16 The B(blue) element of current color is decreased by sixteen.
This menu behavior is same as secondary click tens digit on the detail of color element.
Mask The mask of auxiliary attribute of current color switch to on or off.
This menu behavior is same as click the mask of auxiliary attribute.
Unuse The unuse of auxiliary attribute of current color switch to on or off.
This menu behavior is same as click the unuse of auxiliary attribute.
Transparent The transparent of auxiliary attribute of current color switch to on or off.
This menu behavior is same as click the transparent of auxiliary attribute.
