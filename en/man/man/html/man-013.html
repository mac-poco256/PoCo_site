The File Menu

The file menu relative to file handing such as New, Open, Save and so forth.


New Appear The dialogue panel of new image, which create new docuemnt(image).
Open Open the existece of files.
Open Recent Open recent opend files.
This menu is depend on setting in System Preferences > General > Recent Items.
Close Close front window in the PoCo.app.(include each window.)
Close All Close the whole window in the PoCo.app.(include each window.)
Save Store current document to file.
Save As Store current document to file with specified name.
