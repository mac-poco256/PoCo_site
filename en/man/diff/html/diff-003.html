The difference between versions 1.200(1.2.0) and 1.300(1.3.0)

In be up-to-date version 1.300(1.3.0), the difference between versions 1.200(1.2.0) and 1.300(1.3.0) are listed here.


Add

Added functions are following:

  • Added a function that draw with interpolation curve. As well as, added a function that choose the method of interpolation algorithm.
  • In the integrated setting of auxiliary attributes, a function of the number of color range is added.
  • When drawing the shape of rectangle and the shape of ellipse, a function of specifying rotate angle is added.


Replace, Fix, Correct

Replaced, Fixed, or Corrected functions are following:

  • When a color is specified "not use"(exclude color), the water drop is not use that color.
  • When delete with selection area, apply selected drawing functions. Note that the tint equivalent to the uniformed tint.
  • Replaced the caption of "Erase" to "Delete".
  • Fixed an issue that size of the water drop is greater than size of selection area.


Delete, Deprecate

Deleted, or Deprecated functions are none.
