The difference between versions 1.000(1.0.0) and 1.100(1.1.0)

In be up-to-date version 1.100(1.1.0), the difference between versions 1.000(1.0.0) and 1.100(1.1.0) are listed here.


Add

Added functions are following:

  • The function of the automatic gradient with tile-pattern is added, and that function calling a menu item is also added.
  • The function of the replace coloris added, and that function calling a menu item is also added.
  • The bold width and the thin width menu items are added.
  • The thick tint and the thin tint menu items are added.
  • The function of the copy color is added, and that function calling the button is added, as well as calling a menu itemis added.
  • Specifying the quality of preview for layer preview is added.
  • In the free line or the free line with the pen pressure, equivalent to the eraser drawing function is added whether switch to eraser with the pen or not.
  • The function of the selection with painting is added.


Replace, Fix, Correct

Replaced, Fixed, or Corrected functions are following:

  • All the icons of button had supported for the Retina.
  • The namd of the Bundle Identifier was renamed.
  • Replace the icon of lock color elements.
    An icon represent that either Allow or Disallow.
  • Replace the icon of continuous or discretion.
    An icon represent that either a line or dots.
  • Replace the icon of use of handle.
    An icon represent that either with the handle or without the handle.
  • Replace the icon of selection.
    An icon represent that either free style or rectanlge.
  • Change the caption of Edit(on the table header) to Protect.
  • Fix an issue that patterns of the half air brush which a pattern of size-4 is same as a pattern of size-3.
    In addition, improve a pattern of size-4.


Delete, Deprecate

Deleted, or Deprecated functions are none.
