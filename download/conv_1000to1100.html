1.000 から 1.100 への更新にあたって

補足的な情報（ウィンドウ位置や環境設定の内容など）についてを保存する名称（Bundle Identifier）を変更しました。

そのため、Version 1.100 へ更新してもそれら情報を引き継げません。

必要であれば、ターミナル（Terminal.app）で次のコマンドを実行して UserDefaults の引き継ぎを行ってください。

defaults export jp.harukago.poco jp.harukago.poco.plist

defaults import org.poco256.PoCo jp.harukago.poco.plist

defaults delete jp.harukago.poco

rm jp.harukago.poco.plist

行っている内容は「古い分をファイルに出して」「そのファイルを新しい分に登録して」「古い分を消して」「ファイルに出した分も消す」ということです。


また、レイヤーウィンドウの大きさが変更となりましたので、ターミナル（Terminal.app）で次のコマンドを実行して UserDefaults から値を削除してください。

defaults delete org.poco256.PoCo "NSWindow Frame LayerWindowPos"
